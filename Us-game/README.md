# 🇺🇸 U.S. States Game (Python + Turtle + Pandas)

Ένα διαδραστικό **παιχνίδι γεωγραφίας** όπου ο χρήστης προσπαθεί να μαντέψει όλα τα 50 πολιτείες των Η.Π.Α.  
Χρησιμοποιεί **Python Turtle** για το γραφικό χάρτη και **Pandas** για επεξεργασία δεδομένων.

---

## 🧩 Περιγραφή

- Ο παίκτης βλέπει έναν χάρτη των Ηνωμένων Πολιτειών.
- Πληκτρολογεί ονόματα πολιτειών (case-insensitive).
- Αν το όνομα είναι σωστό, εμφανίζεται στη σωστή τοποθεσία πάνω στον χάρτη.
- Το παιχνίδι τελειώνει όταν ο χρήστης:
  - Βρει και τις 50 πολιτείες, ή  
  - Πληκτρολογήσει **Exit**, οπότε δημιουργείται ένα αρχείο `states_to_learn.csv`  
    με όσες πολιτείες δεν βρήκε.

## Functions

**| Συνάρτηση                         | Περιγραφή                                                  |**
| --------------------------------- | ---------------------------------------------------------- |
| `while len(guessed_states) < 50:` | Κεντρικός βρόχος του παιχνιδιού που ζητά απαντήσεις        |
| `if answer_state == "Exit":`      | Δημιουργεί το αρχείο με τις πολιτείες που λείπουν          |
| `message_turtle.write()`          | Εμφανίζει μήνυμα για διπλές απαντήσεις                     |
| `t.goto()` & `t.write()`          | Τοποθετεί το όνομα της πολιτείας στη σωστή θέση στον χάρτη |
