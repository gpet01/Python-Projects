# ⏰ Pomodoro Timer (Python + Tkinter)

Ένας απλός και λειτουργικός **Pomodoro Timer** φτιαγμένος σε **Python** με **Tkinter GUI**.  
Ακολουθεί την τεχνική **Pomodoro**, που βοηθά στη συγκέντρωση και παραγωγικότητα μέσω χρονικών διαστημάτων εργασίας και διαλείμματος.

---

## 🧠 Τι είναι η τεχνική Pomodoro;

Η μέθοδος **Pomodoro** βασίζεται στην ιδέα ότι η εστιασμένη εργασία για μικρά, καθορισμένα διαστήματα (συνήθως 25 λεπτά)  
με μικρά διαλείμματα (5 λεπτά) αυξάνει την αποδοτικότητα και μειώνει την ψυχική κόπωση.

## 🧩 Features

✅ 25 λεπτά εργασίας (προεπιλογή)  
✅ 5 λεπτά σύντομο διάλειμμα  
✅ 20 λεπτά μεγάλο διάλειμμα (μετά από 4 κύκλους)  
✅ Αυτόματη εναλλαγή μεταξύ εργασίας και διαλειμμάτων  
✅ Οπτική ένδειξη προόδου με ✔️ για κάθε ολοκληρωμένο κύκλο  
✅ Καθαρό και όμορφο GUI σε Tkinter  

---

## 🧰 Requirements

Για να τρέξει το πρόγραμμα χρειάζεσαι:

- **Python 3.x**
- Το αρχείο εικόνας `tomato.png` στον ίδιο φάκελο με το script

> Δεν απαιτούνται εξωτερικά modules — χρησιμοποιεί μόνο τις ενσωματωμένες βιβλιοθήκες `tkinter` και `math`.

---

## 🚀 How to Run

1. Κάνε **clone** το repository:
   ```bash
   git clone https://github.com/<your-username>/<repo-name>.git
   cd <repo-name>
   ```
2. Βεβαιώσου ότι έχεις την εικόνα tomato.png στον ίδιο φάκελο με το main.py.
3. Τρέξε το πρόγραμμα:
   ```bash
   python main.py
   ```
---

## Function

| Συνάρτηση       | Περιγραφή                                                                       |
| --------------- | ------------------------------------------------------------------------------- |
| `start_timer()` | Εκκινεί τον κύκλο Pomodoro (εργασία ή διάλειμμα)                                |
| `reset_timer()` | Επαναφέρει πλήρως το χρονόμετρο και τα checkmarks                               |
| `count_down()`  | Διαχειρίζεται την αντίστροφη μέτρηση και την ενημέρωση της διεπαφής             |

## Ρυθμίσεις 
Μπορείς να αλλάξεις τα προεπιλεγμένα χρονικά διαστήματα στον κώδικα αν θέλεις μικρότερους ή μεγαλύτερους κύκλους:
```bash
WORK_MIN = 25
SHORT_BREAK_MIN = 5
LONG_BREAK_MIN = 20
```
