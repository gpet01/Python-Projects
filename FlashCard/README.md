# 🧠 FlashCard App — Learn Serbian Words 🇷🇸

## 📘 Περιγραφή
Αυτή η εφαρμογή είναι ένα **Flashcard πρόγραμμα** γραμμένο σε Python που βοηθά στην εκμάθηση Σερβικών λέξεων.  
Εμφανίζει μια λέξη στα Σερβικά και, μετά από 3 δευτερόλεπτα, την αντιστοιχία της στα Αγγλικά.  
Ο χρήστης μπορεί να σημειώνει ποιες λέξεις γνωρίζει, ώστε να εστιάζει στις υπόλοιπες.

---

## ⚙️ Λειτουργίες
- Εμφανίζει τυχαία σερβική λέξη.
- Μετά από 3’’ γίνεται αυτόματα “flip” και δείχνει τη μετάφρασή της στα Αγγλικά.
- Ο χρήστης μπορεί να πατήσει ✅ (“γνωστή”) ή ❌ (“άγνωστη”).
- Οι γνωστές λέξεις αποθηκεύονται, ώστε να μην επανεμφανίζονται.
- Τα δεδομένα διαβάζονται/αποθηκεύονται μέσω αρχείων CSV.
